## 概述

个人名片页面采用**黄金圈法则**（Why-How-What）和**4P原则**（Picture-Promise-Prove-Push）设计，通过后台管理系统补充信息即可完美呈现专业形象。

### 页面结构

| 区域 | 原理 | 数据来源 | 作用 |
|------|------|----------|------|
| Hero（头部） | Picture | Profile 基本信息 | 建立第一印象 |
| Why Section | Promise | Profile.bio | 传递核心价值 |
| How Section | Process | Organization 服务信息 | 展示专业能力 |
| What Section | Prove | Case Studies + Honors | 建立信任 |
| CTA Section | Push | Organization CTA | 引导行动 |

## 完整操作流程

### 步骤1：编辑个人资料（Profile）

**目标：** 完善个人基础信息，对应名片的 **Picture（场景画面）** 和 **Why（价值主张）** 部分

#### 1.1 进入编辑页面

```
后台首页 → 侧边栏 "Profiles" → 点击您的名片 → 点击 "Edit" 按钮
```

#### 1.2 填写必填字段

| 字段 | 说明 | 示例 | 前台展示位置 |
|------|------|------|--------------|
| **Full name*** | 您的姓名 | 张伟 | Hero 区域大标题 |
| **Title*** | 职位/职称 | 高级咨询顾问 | Hero 区域副标题 |
| **Company** | 公司名称 | 青狮满天星 | Hero 区域副标题 |
| **Location** | 城市 | 北京 | Hero 区域徽章 |
| **Bio*** | 个人简介 | 见下方撰写建议 | Why Section 主体内容 |

#### 1.3 填写专业领域（Specializations）

**格式要求：** JSON 数组格式

```json
["企业战略咨询", "数字化转型", "团队管理", "商业模式创新"]
```

**展示位置：** Hero 区域徽章（badge-primary样式）

**建议数量：** 3-5个关键词

#### 1.4 上传头像

**方式1：上传文件（推荐）**
- 点击"Avatar"下方的文件选择按钮
- 选择准备好的头像图片
- 建议尺寸：512x512px 正方形
- 支持格式：JPG, PNG, GIF

**方式2：使用外部链接**
- 在"Avatar url"字段粘贴图片URL
- 确保链接可公开访问
- 优先使用方式1，方式2作为备选

#### 1.5 上传背景图片（可选）

- 点击"Background image"下方的文件选择按钮
- 选择高质量横版图片
- 建议尺寸：1920x1080px 或更大
- **展示位置：** Why Section 顶部装饰图

#### 1.6 保存

点击底部 **"Update Profile"** 按钮保存

### 步骤2：配置组织信息（Organization）

**目标：** 完善组织服务信息，对应名片的 **How（方法与能力）** 和 **Push（行动号召）** 部分

#### 2.1 进入编辑页面

```
后台首页 → 侧边栏 "组织设置" → 编辑组织信息
```

#### 2.2 基本信息

| 字段 | 说明 | 前台展示位置 |
|------|------|--------------|
| 组织名称* | 组织/团队名称 | 网站标题、导航栏 |
| 组织描述 | 组织使命简介 | 首页介绍区域 |
| 组织 Logo | Logo 图片 | 导航栏、浏览器图标 |

#### 2.3 服务优势（3组）

**目标：** 回答"我如何帮助您？"，展示差异化优势

**填写区域：** "服务优势" Section

**结构：** 每组包含标题 + 描述

| 优势序号 | 标题示例 | 描述示例 |
|----------|----------|----------|
| 优势1 | 专业团队 | 汇聚行业精英，每位成员都拥有丰富的专业经验和卓越的服务能力 |
| 优势2 | 智能服务 | 24小时AI智能助手，随时为客户解答疑问，智能匹配最合适的专业人士 |
| 优势3 | 品质保证 | 严格的服务标准和质量管理体系，确保每一次服务都超出预期 |

**展示位置：** How Section - 3列网格布局，每列一个优势

#### 2.4 服务流程（4步）

**目标：** 说明服务交付过程，建立信任感

**填写区域：** "服务流程" Section

**结构：** 每步包含标题 + 描述

| 流程步骤 | 标题示例 | 描述示例 |
|----------|----------|----------|
| 步骤1 | 了解需求 | 深入沟通，准确把握您的需求和期望 |
| 步骤2 | 匹配专家 | 智能推荐最合适的团队成员为您服务 |
| 步骤3 | 专业服务 | 提供专业、高效的解决方案 |
| 步骤4 | 持续跟进 | 确保服务质量，持续优化体验 |

**展示位置：** How Section - 带数字徽章的时间线布局

#### 2.5 行动号召（CTA）

**目标：** 引导访客采取行动，促进转化

**填写区域：** "行动号召 (CTA)" Section

| 字段 | 说明 | 示例 |
|------|------|------|
| CTA 标题 | 号召行动的主标题 | 准备好开始了吗？ |
| CTA 描述 | 补充说明 | 无论您有任何问题或需求，我们的专业团队随时准备为您提供帮助 |

**展示位置：** 页面底部，带有"立即咨询"和"了解团队"两个按钮

#### 2.6 保存

点击底部 **"更新组织"** 按钮保存

### 步骤3：添加成功案例（Case Studies）

**目标：** 展示实际成果，对应名片的 **What（成果证明）** 部分

#### 3.1 进入管理页面

```
后台首页 → 侧边栏 "Case studies" → 点击 "New Case study"
```

#### 3.2 填写案例信息

| 字段 | 说明 | 示例 | 是否必填 |
|------|------|------|----------|
| Title* | 案例标题 | 某大型企业数字化转型咨询项目 | 是 |
| Category | 案例分类 | 企业咨询 | 否 |
| Date | 完成日期 | 2024年6月 | 否 |
| Description* | 详细描述 | 为某世界500强企业提供全面的数字化转型咨询服务，帮助其... | 是 |
| Position | 排序位置 | 1, 2, 3... | 否 |
| Profile | 所属名片 | 选择您的 Profile | 是 |

**展示效果：**
- 带序号的案例卡片
- 标题 + 分类标签 + 日期
- 详细描述文本

#### 3.3 添加多个案例

建议至少添加 **3-5个** 成功案例，展示不同类型的服务能力

**排序说明：**
- Position 字段越小，排序越靠前
- 留空则自动递增

#### 3.4 保存

点击 **"Create Case study"** 按钮保存

## 内容撰写建议

### Bio（个人简介）撰写框架

使用黄金圈法则撰写，突出"为什么选择我"：

```
【Why - 使命与信念】（1-2句）
我致力于帮助企业和个人实现数字化转型，相信技术与人文的融合能创造更大价值。

【How - 方法与优势】（2-3句）
拥有15年咨询经验，曾服务50+世界500强企业。擅长战略规划、流程优化和团队赋能，
注重理论与实践结合，为每位客户量身定制解决方案。

【What - 成果与影响】（1-2句）
帮助客户平均提升30%运营效率，培养200+管理人才，获得客户满意度98%好评。
```

**字数建议：** 200-500字

**语气风格：**
- 专业但不生硬
- 自信但不傲慢
- 具体但不冗长

### 服务优势撰写要点

**公式：** 标题（核心卖点） + 描述（具体价值）

**示例对比：**

❌ **弱：** 
- 标题：专业
- 描述：我们很专业

✅ **强：**
- 标题：十年行业经验
- 描述：团队平均从业10年，服务500+企业，深谙行业痛点与最佳实践

**关键要素：**
- 具体数字（10年、500+）
- 客户价值（深谙痛点）
- 差异化优势（最佳实践）

### 服务流程撰写要点

**公式：** 标题（动作） + 描述（价值/方式）

**示例：**

| 步骤 | 标题（动词开头） | 描述（客户视角） |
|------|------------------|------------------|
| 1 | 倾听需求 | 深度访谈，全面了解您的目标与挑战 |
| 2 | 定制方案 | 基于您的实际情况，设计最优解决路径 |
| 3 | 协同执行 | 与您的团队紧密配合，确保落地实施 |
| 4 | 持续优化 | 跟踪效果，持续改进，保证长期成功 |

**避免：**
- 空泛表述（"提供服务"）
- 自我视角（"我们会..."）
- 过度承诺（"100%成功"）

### 成功案例撰写框架

**STAR 法则：**

```
【Situation - 背景】（20%）
某世界500强制造企业面临数字化转型挑战...

【Task - 任务】（10%）
受邀担任首席数字化顾问...

【Action - 行动】（40%）
设计并实施三阶段转型方案：
1. 现状诊断...
2. 流程重构...
3. 系统上线...

【Result - 结果】（30%）
项目成功交付，帮助客户：
• 运营效率提升35%
• 成本降低20%
• 员工满意度提升至85%
```

**关键要素：**
- 可量化的成果（数字、百分比）
- 客户背景（行业、规模）
- 您的角色（顾问、负责人）
- 具体方法（不泄露商业机密的前提下）

### CTA（行动号召）撰写要点

**目标：** 降低决策门槛，鼓励第一步行动

**公式：** 提问 + 承诺 + 行动

**示例：**

✅ **强：**
- 标题：准备好开始了吗？
- 描述：无论您有任何问题或需求，我们的专业团队随时准备为您提供帮助。首次咨询完全免费，让我们一起探索最适合您的解决方案。

❌ **弱：**
- 标题：联系我们
- 描述：如果您有需要请联系我们

**关键要素：**
- 降低门槛（"免费咨询"、"无需承诺"）
- 强调价值（"专业建议"、"定制方案"）
- 明确行动（"点击咨询"、"预约时间"）

## 常见问题

### Q1: 哪些字段是必填的？

**最小可展示配置：**
- ✅ Profile.full_name（姓名）
- ✅ Profile.title（职位）
- ✅ Profile.bio（个人简介）

有了这3个字段，名片就可以基本展示。其他字段越完整，呈现越丰富。

### Q2: 图片尺寸要求是什么？

| 图片类型 | 建议尺寸 | 文件格式 | 文件大小 |
|----------|----------|----------|----------|
| 个人头像 | 512x512px | PNG/JPG | < 500KB |
| 背景图片 | 1920x1080px | JPG | < 2MB |
| 组织 Logo | 512x512px | PNG（透明背景） | < 200KB |

### Q3: 如果某些字段没有填写会怎样？

页面采用**渐进增强**设计，未填写的区块会自动隐藏，不会出现空白：

- ❌ 没有 Bio → Why Section 不显示
- ❌ 没有服务优势 → How Section 不显示
- ❌ 没有案例 → Case Studies 区块不显示
- ❌ 没有荣誉 → Honors 区块不显示
- ❌ 没有 CTA → CTA Section 不显示

### Q4: Specializations 字段格式错误怎么办？

**正确格式：**
```json
["企业战略咨询", "数字化转型", "团队管理"]
```

**常见错误：**
```
❌ 企业战略咨询, 数字化转型（缺少方括号和引号）
❌ ['企业战略咨询', '数字化转型']（使用单引号）
❌ ["企业战略咨询"、"数字化转型"]（使用中文逗号）
```

**修复方法：**
1. 必须使用英文双引号 `"`
2. 必须使用英文逗号 `,`
3. 必须包含在方括号 `[]` 内

### Q5: 如何预览我的名片？

1. 保存所有更改后
2. 访问名片地址：`https://your-domain.com/c/你的slug`
3. 或从后台"Profiles"页面点击名片记录进入详情，再点击"预览"

### Q6: 案例和荣誉的排序如何控制？

**Case Studies 排序：**
- 按 `position` 字段升序（数字小的在前）
- 相同 `position` 按创建时间降序（新的在前）
- 留空则自动递增

**Honors 排序：**
- 按 `date` 字段降序（日期新的在前）
- 相同 `date` 按创建时间降序

**建议：**
- 重要案例设置 position = 1, 2, 3
- 次要案例留空或设置更大数字

### Q7: 多长时间需要更新一次内容？

**建议更新频率：**

| 内容类型 | 更新频率 | 原因 |
|----------|----------|------|
| Bio | 每年1次 | 保持个人定位准确 |
| 案例 | 每季度1次 | 展示最新成果 |
| 荣誉 | 获得后立即添加 | 及时展示新认证 |
| 服务优势 | 每年1-2次 | 优化卖点表达 |
| CTA | 每半年1次 | 测试不同文案效果 |

### Q8: 能否为不同访客显示不同内容？

目前版本暂不支持。所有访客看到相同内容。

**未来规划：**
